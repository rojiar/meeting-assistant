The Approach

Foundation model + linear probe

We do not train a deep network from scratch. WavLM is pre-trained on huge audio data, frozen, and used as a feature extractor.

How Embedding Works

From waveform to vector

inputs = feature_extractor(wav, sampling_rate=16000, return_tensors="pt") frames = wavlm(**inputs).last_hidden_state # (1, time, 768) embedding = frames.mean(dim=1) # mean-pool → (768,)
  • AutoFeatureExtractor converts the raw wave into model inputs.
  • WavLM outputs one hidden vector per time frame.
  • Mean-pooling over time gives a single clip-level embedding.
  • Same embedding feeds both training and inference.
Acoustic Scene Classification7
The Classifier

Linear probe with logistic regression

Training

  • Standardize embeddings (StandardScaler)
  • Fit LogisticRegression on the embeddings
  • Split by recording, not randomly — avoids leakage

Evaluation

  • Group-aware cross-validation
  • Every clip tested on an unseen recording
  • Accuracy, report & confusion matrix

With only ~80 clips, a frozen WavLM + linear head reaches ~91% on held-out recordings — an honest score with no leakage.

Takeaways

What this demo shows

  • Audio foundation models (WavLM / HuBERT) make strong feature extractors out of the box.
  • A linear probe on frozen embeddings is enough for a working scene classifier.
  • Both model and dataset come from the Hugging Face Hub — no manual data prep.
  • The notebook is intentionally simple and explained — a clear teaching/demo artifact.
